HTML 양식





from 요소

HTML <form>요소는 사용자 입력을 위한 HTML 양식을 만드는 데 사용됩니다.

요소 <form>는 텍스트 필드, 체크박스, 라디오 버튼, 제출 버튼 등 다양한 유형의 입력 요소를 위한 컨테이너입니다.

<input> 요소

HTML <input>요소는 가장 많이 사용되는 폼 요소입니다. 속성에 따라 요소 <input>는 여러 가지 방법으로 표시될 수 있습니다 type . 다음은 몇 가지 예입니다.

유형 설명
  • input type="text"
  • input type="radio"
  • input type="checkbox"
  • input type="submit"
  • input type="button"
  • 단일 줄 텍스트 입력 필드를 표시합니다
  • 라디오 버튼(다양한 선택 항목 중 하나를 선택하기 위한)을 표시합니다.
  • 체크박스를 표시합니다(여러 선택 항목 중 0개 이상을 선택하기 위한)
  • 제출 버튼을 표시합니다(양식을 제출하기 위한)
  • 클릭 가능한 버튼을 표시합니다

텍스트 필드

<input type="text">텍스트 입력을 위한 단일 줄 입력 필드를 정의합니다.
