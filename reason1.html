<abbr>

HTML의 <abbr> 요소는 **약어(abbreviation)**를 나타낼 때 사용됩니다.
사용자가 해당 약어에 대해 더 자세한 설명을 알 수 있도록 전체 뜻이나 의미를 제공하는 데 유용합니다.
title 속성과 함께 사용하여 약어의 전체 내용을 명시할 수 있습니다.

주요 특징

1.용도:
  • 약어의 의미를 명확히 전달.
  • 예: "HTML"이라는 약어의 전체 이름 "HyperText Markup Language"를 제공.
2.title 속성
  • 약어의 풀네임 또는 의미를 제공.
  • 사용자가 마우스를 올리면 툴팁으로 표시됩니다.
3.접근성 향상
  • 스크린 리더 등 보조 기술이 약어의 전체 의미를 읽어줌.
  • SEO에도 유리.
4. 브라우저 지원
  • 모든 주요 브라우저에서 지원.
