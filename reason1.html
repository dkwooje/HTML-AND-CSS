Style

HTML의 style 속성은 HTML 요소에 인라인 스타일을 적용하는 데 사용됩니다. CSS(Cascading Style Sheets)를 직접 HTML 요소에 작성하여 해당 요소의 디자인이나 레이아웃을 제어할 수 있습니다.

텍스트 관련

속성 설명 예시
color 텍스트 색상을 설정. color: red;
font-size 텍스트 크기를 설정. font-size: 16px;
font-weight 글씨 두께를 설정. (normal, bold, 숫자값) font-weight: bold;
font-style 글씨 스타일(기울임꼴) 설정. (normal, italic, oblique) font-style: italic;
font-family 글꼴을 설정. font-family: Arial, sans-serif;
text-align 텍스트 정렬. (left, center, right, justify) text-align: center;
text-decoration 텍스트 장식(밑줄, 취소선 등). (none, underline, line-through) text-decoration: underline;
letter-spacing 글자 간격 조절. letter-spacing: 2px;
line-height 줄 간격 조절. line-height: 1.5;
white-space 공백 처리 방식. (normal, nowrap, pre, pre-wrap, pre-line) white-space: pre-wrap;
word-spacing 단어 간격 조절. word-spacing: 10px;

2. 배경 및 색상

속성 설명 예시
background-color 배경색 설정. background-color: #f0f0f0;
background-image 배경 이미지 설정. background-image: url('image.jpg');
background-size 배경 이미지 크기 설정. (cover, contain, auto) background-size: cover;
background-repeat 배경 이미지 반복 설정. (repeat, no-repeat, repeat-x, repeat-y) background-repeat: no-repeat;
background-position 배경 이미지 위치 설정. background-position: center;
background-attachment 배경 고정 여부. (scroll, fixed, local) background-attachment: fixed;
opacity 요소의 투명도 설정. (0부터 1) opacity: 0.5;

3. 박스 모델

속성 설명 예시
width 요소의 너비 설정. (px, %, em, vw, vh 등) width: 100%;
height 요소의 높이 설정. height: 50px;
margin 요소의 외부 여백 설정. (top, right, bottom, left 순서로 작성) margin: 10px 15px;
padding 요소의 내부 여백 설정. padding: 5px 10px;
border 요소의 테두리 설정. (두께, 스타일, 색상) border: 1px solid red;
box-shadow 박스 그림자 설정. (x, y, 흐림 정도, 색상) box-shadow: 2px 2px 5px gray;

4. 플렉스박스(Flexbox) 및 레이아웃

속성 설명 예시
display 요소의 박스 유형 설정. (block, inline, flex, grid 등) display: flex;
justify-content 플렉스 아이템의 정렬 방식 설정. (flex-start, center, space-between) justify-content: center;
align-items 플렉스 아이템의 세로 정렬 방식 설정. (flex-start, center, stretch) align-items: stretch;
flex 플렉스 아이템의 크기 비율 설정. flex: 1;

5. 표 관련 속성

속성 설명 예시
border-collapse 테이블 테두리 결합 여부 설정. border-collapse: collapse;
border-spacing 셀 간의 간격 설정. border-spacing: 10px;
caption-side 테이블 캡션 위치 설정. (top, bottom) caption-side: top;

6. 변환(Transform) 및 애니메이션

속성 설명 예시
transform 요소 변환. (rotate, scale, translate) transform: rotate(45deg);
transition 애니메이션 효과 적용. transition: all 0.3s ease;
animation 복잡한 애니메이션 설정. animation: slidein 3s;
