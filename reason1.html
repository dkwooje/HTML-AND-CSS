Head 요소들

HTML <head>요소는 다음 요소의 컨테이너입니다:
<title>, <style>, <meta>, <link>, <script>, 및 <base>.

<head> : 메타데이터(데이터에 대한 데이터)를 담는 컨테이너로, <html>태그와 <body>태그 사이에 배치됩니다.

요소는 <title>문서의 제목을 정의합니다. 제목은 텍스트 전용이어야 하며 브라우저의 제목 표시줄이나 페이지 탭에 표시됩니다.

<style>요소는 단일 HTML 페이지에 대한 스타일 정보를 정의하는 데 사용됩니다.

<link>요소는 현재 문서와 외부 리소스 간의 관계를 정의합니다. 이 <link> 태그는 외부 스타일 시트에 연결하는 데 가장 자주 사용됩니다.

meta

<meta>요소는 일반적으로 문자 집합, 페이지 설명, 키워드, 문서 작성자 및 뷰포트 설정을 지정하는 데 사용됩니다.

예시:

사용되는 문자 집합을 정의합니다: <meta charset="UTF-8"> 검색 엔진을 위한 키워드 정의: <meta name="keywords" content="HTML, CSS, JavaScript"> 웹 페이지에 대한 설명을 정의하세요. <meta name="description" content="Free Web tutorials"> 페이지의 작성자를 정의하세요: <meta name="author" content="John Doe"> 30초마다 문서 새로 ​​고침: <meta http-equiv="refresh" content="30"> 모든 기기에서 웹사이트가 잘 보이도록 뷰포트 설정: <meta name="viewport" content="width=device-width, initial-scale=1.0">

뷰포트 설정

뷰포트는 웹 페이지에서 사용자가 볼 수 있는 영역입니다. 기기에 따라 다릅니다. 모바일 폰에서는 컴퓨터 화면보다 더 작습니다. <meta>모든 웹 페이지에 다음 요소를 포함해야 합니다 .

<meta name="viewport" content="width=device-width, initial-scale=1.0">

HTML <base> 요소

해당 <base>요소는 페이지의 모든 상대 URL에 대한 기본 URL 및/또는 대상을 지정합니다. 태그 <base>에는 href 또는 target 속성이 둘 다 있어야 합니다. <base> 문서에는 단 하나의 요소만 포함될 수 있습니다 !

예시:

<head> <base href="https://www.w3schools.com/" target="_blank"> </head>
