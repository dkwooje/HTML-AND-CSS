<pre>

HTML의 <pre> 태그는 **"preformatted text"**를 나타내며, 입력된 텍스트를 작성된 그대로 브라우저에 렌더링합니다.
여기에는 공백, 줄바꿈, 탭 등이 포함됩니다. 기본적으로 고정 폭 글꼴(monospaced font)이 적용되며, 텍스트 포맷팅이 유지됩니다.

주요 특징

1.공백과 줄바꿈 유지

<pre> 태그 안에 작성된 텍스트는 HTML에서 기본적으로 무시되는 공백과 줄바꿈을 그대로 유지합니다. 작성한 텍스트의 포맷이 화면에 그대로 나타납니다.

2.고정 폭 글꼴 (Monospace Font)

대부분의 브라우저에서 <pre> 태그 내부 텍스트는 기본적으로 고정 폭 글꼴(예: Courier)로 표시됩니다.

3.자동 줄바꿈 없음

<pre> 태그 내부의 텍스트는 너비를 넘어가는 경우 자동으로 줄바꿈되지 않습니다. CSS를 사용해 제어할 수 있습니다.

기본 사용 예제

                    This    is    preformatted
                    text.
                    It preserves line breaks, 
                            tabs, and spaces.

This is preformatted text. It preserves line breaks, tabs, and spaces.

HTML 코드 표시 예제

코드 조각을 표현할 때 유용합니다.

                <html>
                  <head>
                    <title>Example</title>
                  </head>
                  <body>
                    Hello, World!
                  </body>
                </html>

CSS를 사용한 스타일링

<pre> 태그의 기본 동작을 변경하거나 추가 스타일을 적용할 수 있습니다.

고정 폭 글꼴 변경

                Preformatted text with custom font.

텍스트 자동 줄바꿈 활성화

                This is a very long line of text that will wrap to the next line when it reaches the end of the container.

스크롤바 추가

                Long preformatted text that requires scrolling to view everything.

비교: pre vs code

: 텍스트 포맷팅을 그대로 유지합니다. 주로 코드 블록을 포함하거나, 
                    고정된 레이아웃의 텍스트를 표시하는 데 사용됩니다.
                : 한 줄 코드나 코드 조각을 인라인으로 표시할 때 사용됩니다. 
                    포맷팅은 유지되지 않습니다.
