HTML 반응형 웹 디자인

반응형 웹 디자인은 다양한 화면 크기와 뷰포트에 맞게 자동으로 조정됩니다.

뷰포트 설정

<head>의 다음과 같은 코드가 반응형 웹을 만듭니다.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

반응형 이미지

CSS width속성을 100%로 설정하면 이미지가 반응하여 확대/축소됩니다.

브라우저 폭에 따라 다른 이미지 표시

HTML <picture>요소를 사용하면 다양한 브라우저 창 크기에 맞게 다양한 이미지를 정의할 수 있습니다.

반응형 텍스트 크기

텍스트 크기는 "vw" 단위로 설정할 수 있는데, 이는 "뷰포트 너비"를 의미합니다.
