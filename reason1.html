HTML <small> 요소

HTML의 요소는 텍스트를 작게 표시하기 위해 사용됩니다. 시각적으로 텍스트 크기를 줄이지만, 의미론적으로는 덜 중요한 정보나 부가적인 내용을 나타냅니다. (부가 정보나 법적 고지, 저작권 정보 등 덜 중요한)

이 제품은 14세 이상 사용을 권장합니다.

HTML <mark> 요소

HTML의 <mark> 요소는 텍스트를 **하이라이트(강조)**하는 데 사용됩니다. 이 요소로 감싼 텍스트는 일반적으로 노란색 배경색으로 표시되며, 강조 표시를 통해 중요하거나 관련성이 높은 정보를 시각적으로 돋보이게 합니다.

검색 결과: HTML은 웹 페이지 구조를 정의합니다.

다음 시험 날짜는 2024년 12월 1일입니다.

이 문장은 강조된 텍스트를 포함합니다.

HTML <del> 요소

HTML의 <del> 요소는 삭제된 텍스트를 나타낼 때 사용됩니다. 시각적으로는 취소선(strike-through)이 텍스트에 그려지며, 해당 텍스트가 문서에서 더 이상 유효하지 않음을 나타냅니다.

이 제품의 가격은 ₩50,000 입니다.

수정 전: 이 문장은 잘못되었습니다. 수정 후: 이 문장은 올바릅니다.

이 문서는 2023년 정책을 대체합니다.
