자바스크립트

JavaScript는 HTML 페이지를 더욱 동적이고 대화형으로 만들어줍니다.

HTML <script> 태그

HTML <script>태그는 클라이언트 측 스크립트(JavaScript)를 정의하는 데 사용됩니다.
요소 <script>는 스크립트 문을 포함하거나 src속성을 통해 외부 스크립트 파일을 가리킵니다.
JavaScript의 일반적인 용도로는 이미지 조작, 양식 검증, 콘텐츠의 동적 변경 등이 있습니다.
HTML 요소를 선택하기 위해 JavaScript는 가장 자주 다음 document.getElementById()방법을 사용합니다.
