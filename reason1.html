<ins> '삽입' 태그

HTML의 <ins> 태그는 "insertion"을 의미하며, 문서에 새로 삽입된 텍스트를 나타낼 때 사용됩니다.
일반적으로 추가된 텍스트를 강조하거나 추적 변경 기록을 표시하는 데 사용됩니다.
브라우저에서 기본적으로 삽입된 텍스트는 밑줄로 표시됩니다.

주요 특징:

용도: 문서에서 새로 추가된 내용을 표시합니다.
기본 스타일: 텍스트에 밑줄이 적용됩니다.
속성:
cite: 변경 사항의 출처를 URL로 지정합니다.
datetime: 텍스트가 삽입된 날짜와 시간을 지정합니다. 형식은 YYYY-MM-DDThh:mm:ss입니다.

예시 : 이 문장은 기존 문서의 내용입니다. 이 부분은 새로 추가된 내용입니다.

CSS로 스타일링:

이 문장은 기존 문서의 내용입니다. 이 부분은 새로 추가된 내용입니다.

유사한 태그:

: 삭제된 텍스트를 나타낼 때 사용합니다. 기본적으로 취소선이 적용됩니다.
: 강조 표시된 텍스트를 나타낼 때 사용하며, 기본적으로 배경색이 강조됩니다.
