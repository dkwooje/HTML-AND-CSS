<sub>

HTML <sub> 요소는 텍스트를 아래 첨자(subscript)로 표시할 때 사용됩니다.
보통 화학식, 수학 공식, 그리고 주석과 같은 상황에서 자주 사용됩니다.
이 태그를 적용하면 텍스트가 기준선 아래로 내려가고 크기가 작아집니다.

주요 특징

1.용도: 화학식 표현: H₂O, CO₂
수학 공식: aₙ, xᵢ
주석 번호: 단어₁, 문장₂
2.스타일링: 텍스트가 기준선 아래로 이동하고, 기본 글자 크기보다 작게 표시됩니다.
3.지원: 모든 주요 브라우저에서 지원됩니다.

기본 예제

화학식: H2O

수학식: an = n + 1

주석: 이 문장은 특별합니다1.

스타일링 예제

화학식: H2O

수학식: an = n + 1
