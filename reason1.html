HTML 테이블

HTML에서

요소는 데이터를 행(row)과 열(column)로 구성된 표 형태로 표시할 때 사용됩니다.
테이블은 데이터를 시각적으로 정리하고 이해하기 쉽게 만들며, 주로 통계 정보, 일정, 데이터 목록 등을 표시하는 데 유용합니다.

기본 구조

제목 1 제목 2 제목 3
데이터 1 데이터 2 데이터 3
데이터 4 데이터 5 데이터 6

주요 요소

<table>
  • 테이블 전체를 정의하는 기본 요소.
<thead>
  • 테이블의 머리 부분을 정의.
  • 주로 제목 행(th)을 포함.
<tbody>
  • 테이블의 본문 데이터를 정의.
<tfoot>
  • 테이블의 하단(총계 등)을 정의.
<tr>(Table Row)
  • 행(row)을 정의.
<th>(Table Header)
  • 열의 제목이나 헤더를 정의.
  • 기본적으로 굵은 글씨와 가운데 정렬로 표시.
<td>(Table Data)
  • 셀(cell)에 데이터를 정의.
  • 기본적으로 왼쪽 정렬.
