HTML CSS

HTML에서 CSS는 **스타일링(디자인)**을 적용하여 웹페이지를 시각적으로
더 매력적이고 사용자 친화적으로 만드는 데 사용됩니다.
CSS는 HTML 요소에 색상, 글꼴, 크기, 간격, 배치 등을 정의하고,
다양한 애니메이션 효과와 반응형 디자인도 가능하게 합니다.

1.인라인 CSS

A Blue Heading

A red paragraph.


2.내부 CSS

This is a heading

This is a paragraph.

3.외부 CSS

헤드부분에 link로 하여금 외부 css파일에서 스타일 데이터를 가져온다. 

                    <head>
                      <link rel="stylesheet" href="styles.css">
                    </head>
